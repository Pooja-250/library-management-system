Project Overview

The Library Management System is a Java-based application designed to manage library operations such as book management, member management, and issue/return functionality. It is built using Object-Oriented Programming (OOP) principles and provides a structured way to handle library records efficiently.

🎯 Objectives
Automate library operations
Maintain book and member records
Simplify issue and return process
Practice Java OOP concepts in a real project


Technologies Used
Java
Object-Oriented Programming (OOP)
Java Swing (for GUI, if used)
File Handling / Collections Framework

🚀 How to Run the Project 

Clone the repository:
git clone https://github.com/Pooja-250/library-management-system.git
Open the project in VS Code / IntelliJ IDEA
Navigate to src folder
Compile and run:
javac Main.java
